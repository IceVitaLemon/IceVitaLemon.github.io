Git配置文件路径

三个层次中每层的配置（系统、全局、本地）都会覆盖掉上一层次的配置

系统级配置文件

  • 作用范围：对系统中所有用户和仓库有效

  • 路径：

    • Windows：Git安装路径/etc/gitconfig
    • Linux：/etc/gitconfig

  • 查看配置：git config --system --list

全局配置文件

  • 作用范围：当前用户下的所有 Git 仓库

  • 路径：

    • Windows：C:\Users\<用户名>\.gitconfig
    • Linux：~/.gitconfig 或 ~/.config/git/config

  • 查看配置：git config --global --list

本地配置文件

  • 作用范围：仅对当前仓库有效

  • 路径：

    • Windows/Linux：当前仓库根目录下的.git/config

  • 查看配置：git config --local --list
